# BookManagerWeb

## 简介

这是一个由Java语言编写的一个图书管理系统

## 所用技术

```
Java后端： JDK8、Servlet 5、Thymeleaf 3.0.12
工具库：lombok 1.18.24
数据库：MySql5.7 + Mybatis 3.5.7
测试：Junit 5
前端：bootstrap、Jquery （模板）
```
